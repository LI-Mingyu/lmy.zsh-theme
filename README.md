# lmy.zsh-theme
一个简洁的oh-my-zsh主题，适用场景：
- 需要区分不同服务器（例如区分本地环境和云主机）
- 编程工作为主（git promote）
- 经常熬夜忘记时间
- 显示紧凑（有时候空间有限，比如在vscode的terminal中）
